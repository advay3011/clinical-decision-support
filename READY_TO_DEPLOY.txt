QUICK START (3 STEPS)
================================================================================

1. PUSH TO GITHUB
   $ git add .
   $ git commit -m "Initial commit: Clinical Decision Support Chatbot"
   $ git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
   $ git branch -M main
   $ git push -u origin main

2. DEPLOY TO STREAMLIT CLOUD
   - Go to share.streamlit.io
   - Sign in with GitHub
   - Click "New app"
   - Select your repo, branch "main", file "streamlit_app.py"
   - Click "Deploy"

3. ADD API KEY
   - Click ‚ãØ ‚Üí Settings ‚Üí Secrets
   - Add: ANTHROPIC_API_KEY = "your-api-key"
   - Save

Your app will be live at:
https://YOUR_USERNAME-REPO_NAME.streamlit.app
